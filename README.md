# Automation
A collection of scripts and code used for automation and to automate tasks or workflows.

+ [Page Status](/page-status/) - A tool that checks the HTTP status of page URLs listed in the `sitemap.xml` file and return and `404 Not Found` HTTP status. Written in `Python`.
+ [Uptime Checker](/uptime-checker/) - A tool that checks whether a `website`, `API`, or `server` is online and responding properly. Written in `Python`.
